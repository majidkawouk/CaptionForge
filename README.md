# CaptionForge

Professional animated caption overlays for your videos. Built for content creators who need premium, word-by-word animated captions with transparent backgrounds.

## What is CaptionForge?
CaptionForge is a powerful, privacy-first web application designed to streamline the creation of high-quality caption overlays. It allows creators to generate styled, animated text that can be dropped directly onto any footage in professional video editors.

## Key Features
- **Transparent Export**: Generate `.webm` videos with a transparent alpha channel, ready to drop into any editor (Premiere Pro, DaVinci Resolve, CapCut).
- **Infinite Style Controls**: 
  - Custom Fonts (Google Fonts library)
  - Box Mode (Solid backgrounds for better readability)
  - Skew/Slant controls
  - Advanced stroke, shadow, and letter spacing options
- **Professional Presets**: Instant one-click styles inspired by top creators (MrBeast, Hormozi, etc.).
- **Live Preview**: Real-time rendering of your animations before you export.
- **Privacy First**: 100% client-side processing. Your text and footage never leave your browser.

## Tech Stack
- **Framework**: Next.js
- **Processing**: FFmpeg.wasm (Client-side WASM encoding)
- **Rendering**: HTML5 Canvas (Dynamic frame generation)
- **Styling**: Tailwind CSS (v4.0)
